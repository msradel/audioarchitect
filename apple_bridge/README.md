# Apple Music Bridge (Skeleton)

This folder will contain the open-source Apple Music Bridge implementation for AudioArchitect.

Planned responsibilities:

- Connect to Apple Music using the Apple Music API and MusicKit where appropriate.[web:327][web:358]
- Read user playlists and basic metadata for users with active Apple Music subscriptions.
- Export/import playlists using the neutral formats in `docs/playlist_format.md`.
- Follow the constraints defined in `docs/apple_bridge_ui.md` and `docs/bridge_lockdown.md`.
